Welcome to txtAlert's documentation!
====================================

txtAlert is a mobile technology tool developed by Praekelt Foundation.
It uses text message (SMS) reminders to improve the health of HIV
positive people, by reminding them when they need to go for their next
visit to the clinic.

Visit http://praekeltfoundation.org/txtalert.html for more information.
